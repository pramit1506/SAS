Database Connection

1.
$db_host = "localhost";
$db_user = "root";
$db_pass = ""; // No password as specified
$db_name = "attendance_system";

$db_host: The database server address (localhost = your own computer).
$db_user: The username for the database (root = default for XAMPP).
$db_pass: The password (empty here, as is common in local XAMPP setups).
$db_name: The name of the database to use.

2.
$conn = new mysqli($db_host, $db_user, $db_pass, $db_name);
Creates a new connection to the MySQL database using the above credentials.

3.
if ($conn->connect_error) {
    die("<html>...Error...</html>");
}
If the connection fails, it stops the script and shows a styled error message.

4.
mysqli_report(MYSQLI_REPORT_ERROR | MYSQLI_REPORT_STRICT);
Tells MySQLi to throw exceptions and report errors for easier debugging.

5.
$today = date('Y-m-d');
Gets today’s date in YYYY-MM-DD format (for SQL queries).

6.
$total_students = 0;
$present_today = 0;
$attendance_data = [];

Initializes variables to store total students, number present today, and an array for attendance details.

7.
$result = $conn->query("SELECT COUNT(*) FROM students");
if ($result) $total_students = $result->fetch_row()[0];

Runs a query to count all students.
If successful, stores the count in $total_students.

8.
$stmt = $conn->prepare("SELECT COUNT(DISTINCT roll_number) FROM attendance WHERE DATE(date_time) = ?");
$stmt->bind_param("s", $today);
$stmt->execute();
$result = $stmt->get_result();
if ($result) $present_today = $result->fetch_row()[0];
$stmt->close();

Prepares a query to count unique students who have an attendance record for today.
Binds today’s date as a parameter.
Executes and fetches the result into $present_today.

9.
$sql = "SELECT s.roll_number, s.name, a.date_time
        FROM students s
        LEFT JOIN attendance a ON s.roll_number = a.roll_number AND DATE(a.date_time) = ?
        ORDER BY s.roll_number";
$stmt = $conn->prepare($sql);
$stmt->bind_param("s", $today);
$stmt->execute();
$result = $stmt->get_result();
if ($result->num_rows > 0) {
    while($row = $result->fetch_assoc()) {
        $attendance_data[] = $row;
    }
}
$stmt->close();

Prepares a query to get each student’s roll number, name, and their attendance time for today (if any).
Uses a LEFT JOIN so all students are listed, even if absent.
Orders by roll number.
Fills $attendance_data with each row.

15.
<div class="attendance-list">
    <h2>Today's Attendance (<?php echo date('d-M-Y', strtotime($today)); ?>)</h2>
    <table>
        <thead>
            <tr>
                <th>Roll Number</th>
                <th>Name</th>
                <th>Time</th>
                <th>Status</th>
            </tr>
        </thead>
        <tbody>
            <?php
            if (!empty($attendance_data)) {
                foreach($attendance_data as $row) {
                    $status = $row['date_time'] ? "Present" : "Absent";
                    $time = $row['date_time'] ? date('h:i:s A', strtotime($row['date_time'])) : " - ";
                    echo "<tr>";
                    echo "<td>" . htmlspecialchars($row['roll_number']) . "</td>";
                    echo "<td>" . htmlspecialchars($row['name']) . "</td>";
                    echo "<td>" . $time . "</td>";
                    echo "<td class='" . strtolower($status) . "'>" . $status . "</td>";
                    echo "</tr>";
                }
            } else {
                 echo "<tr><td colspan='4'>No students found or no attendance data for today.</td></tr>";
            }
            ?>
        </tbody>
    </table>
</div>

Table header: Roll Number, Name, Time, Status.
For each student, shows:
Roll number and name (escaped for security).
Time of attendance (if present, formatted as hh:mm:ss AM/PM; otherwise, a dash).
Status: "Present" (if attended) or "Absent".
The status cell gets a class for coloring (see CSS).
If no data, shows a message.
